**Expense Tracker (Python)**<br>
This is a simple Python-based Expense Tracker project that helps users record and manage their daily expenses through a console application. The project is created to practice Python fundamentals and logical problem-solving.

**Features**<br>
Add daily expenses<br>
View all expenses<br>
Calculate total expense<br>
Simple and beginner-friendly<br>

**Technologies Used**<br>
Python 3
